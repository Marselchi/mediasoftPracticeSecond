## Описание проекта

Проект представляет собой REST API турнирной таблицы футбола.

### Основные возможности:
1. **Управление командами**: Создание, редактирование, просмотр и удаление команд. 
(технически не входило в тз)
2. **Управление матчами**: Регистрация матчей, обновление счетов, изменение статусов (Запланирован/Завершен), удаление матчей. (тут полный круд так же не в тз был)
3. **Турнирная таблица**: Динамический расчет турнирного положения команд на основе результатов завершенных матчей.

## Доступы
### База данных 
В тестовой среде
используется Testcontainers, база данных поднимается автоматически со следующими параметрами, прописаными в тесте:

    Database: testdb
    Username: test
    Password: secret

При старте бд применяет миграции liquibase:
```
spring.liquibase.enabled=true
```
### API Доступы
Приложение не имеет встроенной системы аутентификации т.к по тз такого не требовалось

## Swagger UI
После запуска приложения документация API доступна по адресу:
http://localhost:8080/swagger-ui.html

## Как запустить проект
### 1. Запуск через Maven

Для сборки и запуска приложения выполните:
```
mvn clean install
mvn spring-boot:run
```

### 2. Запуск через IDE
   1. Откройте проект в IDE
   2. Дождитесь импорта зависимостей Maven.
   3. Найдите класс с аннотацией @SpringBootApplication (Application.java)
   4. Нажмите кнопку Run или Debug.


### Проверка работы
   После запуска сервера вы можете проверить работоспособность через Swagger 
   

### Тестирование
Проект покрыт тестами двух уровней:

1. Юнит-тесты:

        MatchServiceTest: Проверяет логику создания, обновления и удаления матчей, включая валидацию.
        TeamServiceTest: Проверяет CRUD операций над командами и уникальность имен.
        StandingsServiceTest: Проверяет получение сетки с сортировкой.

2. Интеграционные тесты:

        ChampionshipApiIT: Полный цикл проверки работы API.
            Создает команды и сезон.
            Регистрирует несколько матчей со статусом FINISHED.
            Проверяет, что эндпоинт /standings возвращает корректно отсортированную таблицу с правильным количеством очков.
